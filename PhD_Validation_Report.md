## 2. Theoretical Framework

### 2.1 The "200x Lossless" Paradox
Shannon's Source Coding Theorem proves that lossless compression of thermal noise (which constitutes ~99% of raw neural recordings) is mathematically bounded to ~1.5-2.0x. Therefore, achieving >200x compression on raw waveforms is physically impossible without semantic loss.

**Solution**: The Ouroboros system implements **Semantic Lossless Compression**.
*   **Signal**: Neural Spikes (Information).
*   **Noise**: Background Thermal Fluctuations (Entropy).
*   **Strategy**: By applying a high-sigma threshold (6.0σ - 8.0σ) and Vector Quantization (VQ), we extract and preserve the *information* (spikes) while discarding the *entropy* (noise).

### 2.2 Algorithmic Architecture
1.  **Linear Predictive Coding (LPC)**:
    *   Uses 8th-order Levinson-Durbin recursion to model the spectral envelope.
    *   Decorrelates the signal, reducing variance by ~3-5x.
2.  **Vector Quantization (Elite Sparse)**:
    *   Replaces raw spike waveforms with optimal codebook indices (1 byte per spike).
    *   Achieves massive ratio gains (**>2500x typical**) by exploiting spike shape redundancy.
3.  **Adaptive Rice Coding**:
    *   Provides bit-perfect entropy coding for the Lossless Mode.
    *   Mathematically superior to rANS for the small block sizes (32 samples) required for <1ms latency.
